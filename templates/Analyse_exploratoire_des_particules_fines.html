Dans ce blog, nous ferons des analyses sur les particules fines (PM2,5), qui sont des polluants qui existe dans lair ambiant et qui sont trs nfastes pour la sant humaine. Ces donnes prsentes ont t publi par lEnvironmental Protection Agency (EPA), qui publie sa base de donne sur les missions de (PM2,5) tous, les 3 ans et connu sous le nom dInventaire national des missions (NEI).

Pour plus dinfos sur le NEI voir NEI.

Les donnes que nous utiliserons pour effectuer nos analyses sont celles de 1999, 2002, 2005, 2008. et peuvent tre tlcharger. ici: donne.

Le contenu du fichier zip:

1) summarySCC_PM25.rds: Base de donne avec les donnes sur les missions de PM2,5 pour les annes 1999, 2002, 2005 et 2008. Le tableau contient un nombre de tonnes de PM2,5 mis par un type de source spcifique pendant toute lanne.

Variables: - fips: Comt amricain. - Pollutant: Le polluant. - SCC : Le nom de la source. - Emissions: Quantit de PM2,5 mise. - Anne : Anne des missions enregistres. - Type : Le type de source.

2) Source_Classification_Code.rds: Mappage des chanes de chiffres SCC dans la table des missions au nom rel de la source PM2,5.
