Machine Learning : Régression Logistique

  • x = cbing(x_train,y_train)
  • Entraînement des modèles en utilisant les données d’entraînement et vérifier
  • Score
  • logistique = glm(y_train~., data = x, family = ‘binomial’)
  • summary(logistique)
  • Effectuer la prediction sur les données de test = predict(logistique,x_test)
